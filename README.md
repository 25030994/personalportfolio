KHODANI's Personal Portfolio

A modern, responsive personal portfolio website showcasing skills, projects, and experience in web development and HCI.

Features

- Responsive Design: Optimized for all devices
- Dark/Light Theme: Toggle between themes with local storage
- Smooth Animations: Scroll-triggered animations and hover effects
- Interactive Elements: Contact form, typing effect, and mobile menu
- Modern UI: Clean design with gradients and shadows
- SEO Friendly: Proper HTML structure and meta tags

Sections

1. Home: Hero section with introduction and call-to-action buttons
2. About: Personal background, stats, and profile image
3. Skills: Categorized skills in Frontend, Design, and Tools
4. Projects: Showcase of key projects with images and links
5. Contact: Contact information and functional contact form

Technologies Used

- HTML5: Semantic markup and accessibility
- CSS3: Modern styling with CSS Grid, Flexbox, and animations
- JavaScript: Interactive features and DOM manipulation

File Structure

personal-portfolio/

├── css/
│   └── styles.css      # All styling
├── js/
│   └── script.js       # JavaScript functionality
├── images/             # Image assets
└── README.md           # This file

Getting Started

1. Clone or download the project files
2. Open index.html in a modern web browser
3. Customize the content, images, and links as needed
4. For production, minify CSS and JS files

Customization

- Replace placeholder images in the images/ folder
- Update personal information in index.html
- Modify colors and styles in css/styles.css
- Add or remove sections as needed

Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
